Lists in html

There are three types of lists in html

  1. ordered list:
    ordered list items are markded with numbers by default.
    For example:
    1. hi
    2. hello
    3. hey
  2. unordered list:
    unordered list items are marked with bullet points by default.
    For example:
    • tea
    • juice
    • milk
  3. description list:
    a description list is a list of terms with description of each term.
    For Example:
    1. coffee-
      black cold drink
    2. milk-
      white cold drink
    1. coffee-
      black cold drink
    2. milk-
      white cold drink
